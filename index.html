Difference between local storage and sesssion storage?

Both local storage and session storage can store a large amount of data and these data is never transferred to server.They are accessible via Js and HTML5

the only defference is local storage data has no expiration date.It remains on client's computer even we close the tab or shut down the pc. On the other hand session data expires when the client closes the tab or log out from a website.



How does Javascript works?

Chrome uses v8 engine to run javascript.At first the engine takes the JS source code and parse it by a parser and makes an abstract syntax tree.Then V8 engine's interpreter generates bytecode(which machine can read ) from the syntax tree by interpreting each line .Then the machine execute the bytecode.For better performance the browser uses an internal technology called just-in-time(JIT) compilation alongside,which compiles javascript into bytecodes at runtime .V8 engine also runs an optimazation compiler parallely to make high optimized machine code by replacing section of bytecodes.
